## 🚀 Usage Guide

### **Start Enhanced CLI**

```bash
# Default interactive mode
forsure

# Explicit interactive mode
forsure interactive
forsure ui

# Check version
forsure --version
```

### **First-Time Experience**

1. **Launch**: `forsure` - Shows large FORSURE ASCII art
2. **Welcome Messages**: Bot greets with helpful tips
3. **Panel Focus**: Chat panel active by default
4. **Status Bar**: Shows "Active: Chat | Uptime: 0:01"

### **Enhanced Chat Commands**

```
help                    # Comprehensive command guide
generate test-demo.forsure  # Parse our demo file
validate test-demo.forsure   # Validate with statistics
list                   # Find all .forsure files
status                 # Live session statistics
about                  # CLI information
theme blue             # Switch to ocean theme
save demo-session       # Save conversation
export                 # Export to JSON
```

### **Enhanced Terminal Commands**

```
ls                     # Directory listing with emojis
pwd                    # Current working directory
echo "Hello ForSure!"   # Text output
date                   # Current timestamp
whoami                 # Current user
mkdir new-project       # Create directory (simulated)
touch app.js           # Create file (simulated)
help                   # Terminal command help
```

### **Professional Keyboard Shortcuts**

- **Tab**: Switch panels with visual feedback
- **F1**: Comprehensive help modal (90% screen)
- **F2**: Clear all panels instantly
- **F3**: Save session with timestamp
- **F4**: Export history to JSON file
- **Ctrl+C**: Exit confirmation dialog

## 🎨 Theme System

- **dark** (default): GitHub-inspired (#0d1117)

### **Theme Switching**

